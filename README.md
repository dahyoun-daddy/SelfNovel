## 기능

### 자기소개서 게시판
- 자기소개서 조회
  - 첨삭 기능
  - PPT 미리보기 기능
  - 신고 기능
  - 추천 기능
  - 자기소개서 EXCEL 다운

- 자기소개서 작성
  - 작성 기능
  - PPT 업로드 기능


### 전문가 게시판
- 전문가별 프로필 페이지 조회
  - 프로필 페이지
  - 첨삭 신청 기능  

### 마이 페이지
- 일반회원 페이지
  - 회원정보 수정 기능
  - 내가 쓴 글 조회 기능
  - 첨삭 신청 리스트

- 전문가 페이지
  - 회원정보 수정 기능
  - 첨삭 신청 받은 리스트

- 관리자 페이지
  - 회원정보 수정
  - 회원 리스트
  - 신규 관리자 등록
  - 관리자 메시지함

### 로그인
  - 로그인
  - 회원가입

### 랭킹
- 자기소개서 랭킹
- 전문가 랭킹
